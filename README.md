# Jarvis Strategies

AI readiness assessments and implementation for construction and manufacturing companies.

## Website

This repository contains the source code for jarvisstrategies.com

## Features

- Professional consulting website
- Google Calendar integration for availability display
- Automated booking request workflow
- Email-based approval system
- Mobile responsive design
- Animated background

## Deployment

Deployed on Vercel with custom domain jarvisstrategies.com

## Booking System

The site includes a custom booking request system that:
- Shows real availability from Google Calendar
- Collects prospect information via form
- Sends email for manual approval/decline
- Creates calendar events automatically
- Suggests alternate times when needed

See `SETUP_GUIDE.md` for configuration instructions.

## Tech Stack

- HTML/CSS/JavaScript (frontend)
- Vercel Serverless Functions (backend)
- Google Calendar API (availability)
- Nodemailer (email notifications)
