### 3. Push to GitHub
```powershell

# Push to GitHub
git push origin main
```

### 4. Configure Vercel Environment Variables
Go to https://vercel.com → Your Project → Settings → Environment Variables

Add these 6 variables (see SETUP_GUIDE.md for values):
- GOOGLE_CALENDAR_API_KEY
- GOOGLE_CALENDAR_ID
- SMTP_HOST
- SMTP_PORT
- SMTP_USER
- SMTP_PASS

### 5. Wait for Auto-Deploy
Vercel will automatically detect the push and redeploy (takes ~30-60 seconds)

### 6. Test the System
- Visit https://jarvisstrategies.com
- Check if button shows "Next Available: [time]"
- Submit a test booking request
- Check your email for approval link

## Quick Reference: What Each File Does

**get-availability.js** - Checks your Google Calendar and returns next free 30min slot

**submit-request.js** - Processes booking form, sends you approval email

**approve.js** - Creates calendar event, sends confirmation to prospect

**suggest-alternate.js** - Sends alternate time proposal to prospect

**decline.js** - Sends polite rejection email to prospect

**package.json** - Lists Node.js dependencies (googleapis, nodemailer)

**.gitignore** - Prevents committing sensitive files

**.env.example** - Template showing what environment variables are needed
